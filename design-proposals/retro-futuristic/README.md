# RETRO-FUTURISTIC Portfolio Design

## Concept: "DIGITAL FRONTIER"

80s synthwave aesthetic meets philosophical depth. Neon grids, CRT artifacts, terminal aesthetics.

## Visual Language

- **Palette**: Deep purple/black base, cyan/magenta/hot pink neon accents
- **Typography**: VT323 (terminal), Press Start 2P (headings), Space Mono (body)
- **Effects**: CRT scanlines, chromatic aberration, pixel noise, glow pulses
- **Layout**: Asymmetric grids, perspective horizon lines, floating elements

## Philosophy Tie-In

"Where logic meets the digital frontier" - the retrocomputing aesthetic echoes the structured, systematic nature of philosophical logic while the neon chaos represents creative exploration.

## Key Sections

1. **BOOT_SEQUENCE** - Animated intro with terminal boot
2. **PROFILE.exe** - Bio with glitchy avatar
3. **PROJECTS.db** - Grid of cards with hover scanlines
4. **BLOG.log** - Cinema posts as film reels/VHS tapes
5. **CONTACT.sys** - Terminal-style contact form

## Animation Philosophy

Heavy but purposeful: scanlines constant, glows pulse on interaction, glitch effects on transitions.

## Tech

React + TypeScript + CSS custom properties. No external animation libs needed.
