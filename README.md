# all-those-symbols

Personal portfolio site for Walter Andrade. Logic × Philosophy × Code.

## Stack
- React 19 + TypeScript
- Vite + Express
- Tailwind CSS 4
- Framer Motion
- PostgreSQL + Drizzle
- Resend (email)

## Dev
```bash
bun install
bun run dev
```

## Build
```bash
bun run build
bun run start
```

## Structure
```
client/          # React frontend
  src/
    components/  # UI components
    pages/       # Route pages
    lib/         # Utils, data
api/             # Express backend
dist/            # Build output
```
